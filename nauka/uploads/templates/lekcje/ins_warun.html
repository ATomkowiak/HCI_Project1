Instrukcje warunkowe

Czym są?

Instrukcja warunkowa służy do sprawdzenia prawdziwości wyrażenia warunkowego.
W zależności od tego czy wyrażenie warunkowe jest prawdziwe czy nie, wykonany zostanie inny fragment kodu. Przykładowy program sprawdzający czy użytkownik jest pełnoletni.

print ( "Ile masz lat?" )
wiek = int ( input ())
if wiek >= 18:
print ( " Brawo !" )
print ( " Jesteś pełnoletni !" )
else:
print("Nie jesteś pełnoletni")

W języku Python bardzo istotne są wcięcia. W powyższym przykładzie wszystko co poprzedzone jest wcięciem ”należy” do bloku instrukcji ”if”.
Oznacza to, że wszystko co jest poprzedzone wcięciem zostanie wykonane jeśli warunek jest prawdziwy.
Podobnie z instrukcją else, która zostanie wykoanan jeśli warunek przy if nie zostanie spełniony
kolejną instrukcją jest elif (else if) która pozwala na stworzenie większej ilości sprawdzanych warunków.
